Documentation for the Calamity desktop proxy client. This site focuses on setup, day-to-day usage, TUN mode behavior, and repository-level development notes.

What is included

  • Tray workflow for quick connect, disconnect, mode switching, and dashboard access.
  • Node, rule, DNS, subscription, log, and Tailnet pages in the main dashboard.
  • TUN mode notes for macOS, including administrator privileges and Fake-IP behavior.
  • Repository references for local development and manual release packaging.

Pages Deployment

  • This site is deployed from docs/ via GitHub Actions.
  • The workflow is defined in .github/workflows/pages.yml.
  • Pushes to main that modify docs or the Pages workflow trigger a new deployment.
